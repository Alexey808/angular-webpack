# Test project (angular - webpack)

**Запуск проекта**
```
1) npm i
2) npm run dev
```

**Запуск проекта через docker**
```
1) npm i (временно)
2) перейти в папку docker/
3) запустить ./build.sh (только один раз)
4) запустить ./run.sh
```
